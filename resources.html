Resources for Web Development

  • Coolors
    • Coolors is a color scheme generator. If you are like me and need some help figuring out which colors look good together, then look throuh coolors trending palettes for ideas. When you move your mouse over the pallets, the hex code for the color shows up and all you have to do is right click it to save the hex code for the color. It's very user friendly!
